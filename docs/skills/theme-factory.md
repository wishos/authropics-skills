# 主题工厂 (Theme Factory) 指南

## 触发条件

当使用主题为制品（artifacts）设置样式时使用此技能。这些制品可以是幻灯片、文档、报告、HTML 登陆页面等。有 10 个预设主题的配色方案和字体可以应用于已创建的任何制品，或者可以动态生成新主题。

---

## 概述

此技能提供精选的专业字体和颜色主题集合，每个都有精心挑选的配色方案和字体配对。一旦选择主题，它可以应用于任何制品。

## 目的

为演示幻灯片或其他制品应用一致、专业的样式时使用此技能。每个主题包括：
- 带有十六进制代码的内聚配色方案
- 标题和正文的配套字体配对
- 适合不同场景和受众的独特视觉身份

## 使用说明

为幻灯片或制品应用样式：

1. **展示主题展示**：显示 `theme-showcase.pdf` 文件，让用户可以直观地看到所有可用主题。不要对其进行任何修改；仅显示文件供查看。
2. **询问他们的选择**：询问要应用于演示文稿的哪个主题
3. **等待选择**：获取关于选择主题的明确确认
4. **应用主题**：一旦选择了主题，将所选主题的颜色和字体应用于演示文稿/制品

## 可用主题

以下 10 个主题可用，每个都在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths** - 专业且平静的海事主题
2. **Sunset Boulevard** - 温暖活力的日落颜色
3. **Forest Canopy** - 自然沉稳的大地色调
4. **Modern Minimalist** - 简洁当代的黑白
5. **Golden Hour** - 丰富温暖的秋色调色板
6. **Arctic Frost** - 凉爽清新的冬日主题
7. **Desert Rose** - 柔和精致的粉色调
8. **Tech Innovation** - 大胆现代的科技美学
9. **Botanical Garden** - 新鲜有机的花园颜色
10. **Midnight Galaxy** - 戏剧性的深空色调

## 主题详情

每个主题在 `themes/` 目录中有完整定义，包括：
- 带有十六进制代码的内聚配色方案
- 标题和正文的配套字体配对
- 适合不同场景和受众的独特视觉身份

## 应用过程

选择首选主题后：
1. 从 `themes/` 目录读取相应主题文件
2. 在整个演示文稿中一致应用指定的颜色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉身份

## 创建您自己的主题

当现有主题都不适合制品时，创建自定义主题。根据提供的输入，生成与上述类似的新主题。给主题一个描述字体/颜色组合代表什么的类似名称。使用提供的任何基本描述选择适当的颜色/字体。生成主题后，显示供审查和验证。然后按照上述方式应用主题。
