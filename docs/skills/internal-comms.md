# 内部沟通 (Internal Comms) 指南

## 触发条件

当用户要求撰写某种内部沟通（进度报告、领导层更新、3P 更新、公司通讯、常见问题解答、事件报告、项目更新等）时使用此技能。

## 何时使用此技能

撰写内部沟通时，使用此技能用于：
- 3P 更新（进展、计划、问题）
- 公司通讯
- 常见问题解答回复
- 进度报告
- 领导层更新
- 项目更新
- 事件报告

## 如何使用此技能

撰写任何内部沟通：

1. **从请求中识别沟通类型**
2. **从 `examples/` 目录加载适当的指南文件**：
   - `examples/3p-updates.md` - 用于进展/计划/问题团队更新
   - `examples/company-newsletter.md` - 用于公司通讯
   - `examples/faq-answers.md` - 用于回答常见问题
   - `examples/general-comms.md` - 用于任何不完全匹配上述的内容
3. **遵循该文件中关于格式、语气和内容收集的具体说明**

如果沟通类型不匹配任何现有指南，请请求澄清或更多关于所需格式的上下文。
